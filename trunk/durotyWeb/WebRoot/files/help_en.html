How do I use Advanced Search?

Advanced search operators are query words or symbols that perform special actions in Duroty search. These operators allow you to find what you're looking for quickly and accurately. They can also be used to set up filters so you can organize your inbox automatically. Some of the most useful operators are listed below.

Operator Definition Example(s)
from: Used to specify the sender Example - from:amy
Meaning - Messages from Amy
to: Used to specify a recipient Example - to:david
Meaning - All messages that were sent to David (by you or someone else)
subject: Search for words in the subject line Example - subject:dinner
Meaning - Messages that have the word "dinner" in the subject
body: Search for words in the body (text or html) Example - body:dinner
Meaning - Messages that have the word "dinner" in the body (text or html)
attachments: Search for words in the attachments
(word, excel, pdf, txt, etc.) 		Example - attachments:project
Meaning - Messages that have the word "project" in the attachments (word, excel, pdf, txt, etc.)
OR Search for messages matching term A or term B*
*OR must be in all caps 		Example - from:amy OR from:david
Meaning - Messages from Amy or from David
-
(hyphen) 		Used to exclude messages from your search Example - dinner -movie
Meaning - Messages that contain the word "dinner" but do not contain the word "movie"
hasattachments:true
Search for messages with an attachment Example - from:david hasattachments:true
Meaning - Messages from David that have an attachment
filetype: Search for an attachment by name or type Example - filetype:pdf
Meaning - Messages have a PDF file as an attachment
" "
(quotes)
 Used to search for an exact phrase*
*Capitalization isn't taken into consideration 		Example - "i'm feeling lucky"
Meaning - Messages containing the phrase "i'm feeling lucky" or "I'm feeling lucky" Example - subject:"dinner and a movie"
Meaning - Messages containing the phrase "dinner and a movie" in the subject
( )
Used to group words
Used to specify terms that shouldn't be excluded 		Example - from:amy(dinner OR movie)
Meaning - Messages from Amy that contain either the word "dinner" or the word "movie" Example - subject:(dinner movie)
Meaning - Messages in which the subject contains both the word "dinner" and the word "movie"
